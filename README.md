# Emomap-backend

## Как запустить

1. `make start-db`
2. Подождать пока запустится БД
3. `make migrate`
4. `make start`

> Если у вас не установлен `make`, то можно посмотреть соответствующие команды в `Makefile` и выполнить их самостоятельно

Бекенд будет запущен на порту `8080`. Документация будет доступна на `http://localhost:8080/docs`
